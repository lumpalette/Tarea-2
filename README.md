## 📝 Reflexión

En este ejercicio aprendí a organizar contenidos repetitivos de forma profesional usando la etiqueta `article` para agrupar cada noticia de manera independiente. Repase también la jerarquía visual mediante el uso de h1 para el título principal y h3 para los artículos. También aprendí un poco más acerca de las flexboxes y media queries para controlar el orden de los contenedores en diferentes tamaños.
